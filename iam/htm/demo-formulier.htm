Opslaan (s):

  • Klik op de 's' knop om je antwoorden op te slaan
  • De data wordt opgeslagen in de localStorage van je browser
  • Als je bent ingelogd, wordt de data ook gekoppeld aan je account

Herstellen (r):

  • Klik op de 'r' knop om eerder opgeslagen antwoorden te herstellen
  • Het formulier wordt automatisch ingevuld met je opgeslagen data

Privacy:

  • Alle data blijft lokaal op jouw apparaat
  • Niets wordt naar een server gestuurd
  • Je hebt volledige controle over je eigen gegevens

Tips:

  • Sla regelmatig op tijdens het invullen
  • Print het formulier voor je dossier
  • Maak een backup via de hoofdpagina
