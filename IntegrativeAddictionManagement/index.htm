ℹ️ Informatie

Over dit project

Dit project bevat verschillende tools en formulieren voor zelfmanagement bij verslavingsproblematiek. Het is ontwikkeld om mensen te helpen zelfstandig aan hun verslaving te werken.

Technische details

  • Xalcohol-htm: Moderne client-side versie met localStorage (geen server nodig)
  • htm-version: Originele formulieren met cookie opslag
  • Xalcohol: Legacy folder met resources (PHP verwijderd)

Privacy

Alle data in de nieuwe Xalcohol-htm versie wordt 100% lokaal op jouw apparaat opgeslagen. Niets wordt naar een server gestuurd.
