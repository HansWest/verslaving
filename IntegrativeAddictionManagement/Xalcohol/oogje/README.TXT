HI GUYSthanks for downloading my php organizer. in this script you can keep track of your appointments, bookmarks, contacts, pop email, ftp files and notes. it is designed to be small, to the point and useful. the email and ftp section are based upon the onefile scripts from www.bonneville.nl. INSTALLINGupload all files to your ftp server. then change the permissions of the directory so everybody can write to this. Do the same for all txt files. This setting of permissions -chmod- is a standard feature of all ftp programs. (it is usually found with a right-click on the file).now everything works except the email and ftp scripts. To make them work you have to open the files email.php and files.php in a text editor and right at the top of the files insert your email/ftp account name, username and password, replacing the dummy text i have put in there.SOME HINTSyou should protect the directory of the organizer. the best option is to make it a protected directory. in mostcases this is a feature that can be found in the control panels of your webserver. if you don't have it included in your webserver software try searching for 'htaccess' at www.hotscripts.com. the file header.php includes html headers and a style sheet that applies to all pages. if you open and edit this file, it's easy to adjust the look of the pages. it's all plain html, so don't be afraid to mess up anything.the file index.php only has one line that refers to the page you want backdoor to start up with. right now it isnotes.php but you can easily change this in a text editor.some people might not get the email script to work properly. the script uses the imap_open command to open a pop3 mailbox. this command has a lot of parameters. if you're a bit into php, look at www.php.net for imap_open and then try some alternative settings for the email script. the imap_open function is only used once, right at the top of the script.RECENT CHANGES2.0: re-written all textfile-based pages.LICENSINGthis script is licensed under the GNU-GPL License (http://www.gnu.org/copyleft/gpl.html).
