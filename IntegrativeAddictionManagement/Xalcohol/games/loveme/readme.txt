With this little game in Javascript: you can play
the electronic daisy to see if the person you love 
thinks about you.

The download of this script is free of charge.
It can be used freely on private or commercial sites.
This script is free software, distributed under the terms of the GNU license.


1) Unzip the files  with a software such as Winzip.

2) Upload all script files into a directory of your choice.

3) The script is now ready to use and you can play this game
   from the page index.html.

4) You can use it through the web or standalone
